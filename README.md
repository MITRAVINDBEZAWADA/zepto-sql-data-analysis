# 🛒 Zepto SQL Data Analysis Project

## 📌 Project Overview

This project focuses on analyzing the **Zepto Inventory Dataset** using **PostgreSQL**. The objective is to simulate the work of a Data Analyst by importing raw e-commerce data, cleaning it, exploring the dataset, and generating meaningful business insights using SQL.

The project demonstrates the complete data analysis workflow, from handling raw data to answering real-world business questions.

---

## 🎯 Project Objectives

* Import a real-world e-commerce dataset into PostgreSQL.
* Perform data exploration to understand the dataset.
* Clean inconsistent and missing data.
* Analyze product inventory using SQL.
* Generate business insights to support decision-making.
* Build a portfolio-ready SQL project.

---

## 📂 Dataset

* **Dataset Name:** Zepto Inventory Dataset
* **Source:** Kaggle
* **Dataset Link:** https://www.kaggle.com/datasets/palvinder2006/zepto-inventory-dataset

> **Note:** The dataset is not included in this repository. Please download it directly from Kaggle using the link above.

## 📊 Project Workflow

### 1. Database Creation

* Created a new PostgreSQL database.
* Designed the table structure for storing inventory data.

### 2. Data Import

* Imported the CSV dataset into PostgreSQL.
* Verified successful data loading.

### 3. Data Exploration

Performed exploratory analysis to understand:

* Number of products
* Product categories
* Missing values
* Duplicate records
* Pricing information
* Inventory details

### 4. Data Cleaning

Cleaned the dataset by:

* Removing invalid records
* Handling missing values
* Correcting zero-price entries
* Converting prices from paise to rupees where required

### 5. Business Analysis

Solved business-related questions using SQL, including:

* Product distribution across categories
* Highest and lowest priced products
* Inventory availability
* Revenue opportunities
* Discount analysis
* Stock insights

## 🚀 How to Run This Project

1. Install PostgreSQL and pgAdmin.
2. Create a new database.
3. Create the required table using the SQL script.
4. Download the dataset from Kaggle.
5. Import the CSV file into PostgreSQL.
6. Execute the SQL queries provided in `zepto_analysis.sql`.
7. Review the outputs and business insights.
