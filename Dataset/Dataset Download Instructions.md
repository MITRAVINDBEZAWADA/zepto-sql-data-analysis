# Dataset Download Instructions

This project uses the **Zepto Inventory Dataset** from Kaggle.

The dataset is not included in this repository.

Download it from:

https://www.kaggle.com/datasets/palvinder2006/zepto-inventory-dataset

Steps:

1. Download the ZIP file from Kaggle.
2. Extract the ZIP file.
3. Open the extracted folder.
4. Use the file named `zepto_v2.csv`.
5. Import it into PostgreSQL using pgAdmin.
